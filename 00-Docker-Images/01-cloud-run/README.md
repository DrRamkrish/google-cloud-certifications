# Docker Image


## Step-01: V1: Build, Tag and Push Docker Image
```t
# Build Docker Image
docker build -t stacksimplify/google-cloud-run:v1 .

# Push Docker Image
docker login
docker push stacksimplify/google-cloud-run:v1
```
